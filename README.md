# 🚀 Evolução em Lógica de Programação e Python

Este repositório documenta minha jornada de aprendizado, desde os fundamentos de lógica até a aplicação prática em linguagens de mercado, como parte da minha graduação em **ADS**.

---

## 🏗️ Fase 1: Fundamentos com Portugol (Exercícios 01 ao 16)
Nesta etapa inicial, utilizei o **VisualG** para focar exclusivamente no desenvolvimento do raciocínio lógico, sem as complexidades de sintaxe.
* **Foco:** Sequências básicas, entrada/saída de dados e operadores aritméticos.
* **Motivo:** Construir uma base sólida antes de avançar para linguagens de programação reais.

---

## 🐍 Fase 2: Aprendizado de Python (Exercícios 17 ao 25)
A partir do exercício 17, migrei para o **Python 3.12**, focando em **Condições Básicas** (`if/else`) e lógica de decisão.

- **Exer 17:** Cálculo de multa de trânsito.
- **Exer 18:** Verificação de idade para votação.
- **Exer 19:** Média escolar e aproveitamento.
- **Exer 20:** Identificação de números Pares ou Ímpares.
- **Exer 21:** Verificação de Ano Bissexto.
- **Exer 22:** Alistamento militar (tempo restante/atraso).
- **Exer 23:** Promoção de Dia das Mulheres (Descontos diferenciados).
- **Exer 24:** Cálculo de preço de passagem por distância.
- **Exer 25:** Desafio: Verificação de existência de triângulos.

## 🐍 Fase 3: Condições Compostas (Exercícios 26 ao 37)
Nesta etapa, avancei para estruturas de decisão mais complexas utilizando `if/elif/else`, explorando operadores lógicos e validações múltiplas.

* **Exer 26:** Comparação de valores numéricos (Maior, menor ou igual).
* **Exer 27:** Cálculo de média com status (Aprovado, Recuperação ou Reprovado).
* **Exer 28:** Classificação de terrenos por área ($m^2$).
* **Exer 29:** Reajuste salarial baseado em tempo de empresa.
* **Exer 30:** [Desafio] Classificação de triângulos (Equilátero, Isósceles ou Escaleno).
* **Exer 31:** [Desafio] Jogo de JoKenPo (Pedra, Papel e Tesoura).
* **Exer 32:** [Desafio] Jogo de adivinhação de números.
* **Exer 33:** Sistema de aprovação de empréstimo bancário.
* **Exer 34:** Cálculo de IMC e classificação de faixa de peso.
* **Exer 35:** Aluguel de carros (Popular vs Luxo) com preços por Km.
* **Exer 36:** Sistema de pontos e recompensas por atividade física.
* **Exer 37:** Reajuste salarial diferenciado por gênero e tempo de serviço.
